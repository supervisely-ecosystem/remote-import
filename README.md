# remote-import

## Prepare access to your data

`cd to folder`

and then 

`docker run -p 8088:80 -v $(pwd):/mnt/data jetbrainsinfra/nginx-file-listing:0.2`

then go to page in your browser and check that files can be viewed or downloaded 

`http://localhost:8088/`
